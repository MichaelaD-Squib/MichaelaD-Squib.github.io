The HTML tag is used when emphasing text is needed. This will cause the screen reader to read the text with emphasis. The text will also be written in italics. This HTML tag is used when describing important information and will cause the text to be bold. The screen reader will understand that the text is important and share it as such, to the listener. This HTML tag is used to describe superscript text. Superscript text is used for text instances such as dates and measurments, among other things. For example, today is June 10th. This HTML tag is used to underline text. This can be used on text that is a title, header or heading. For example, how I used it in my header of this page. This HTML tag is used to highlight important text. This can be used to help ensure that important information text stands out compared to the rest of the text on the page. For example, the due date is June 16th.
